# H3arn Mytap

## How do I install these formulae?

`brew install h3arn/mytap/<formula>`

Or `brew tap h3arn/mytap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "h3arn/mytap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
